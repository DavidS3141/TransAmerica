This is a debbuger\-A\-I, where you actually can implement code. It exists for everyone, to enter code, that should be, by the rules of the game, not possible. If you found a cheat, then please show here, so that we can try to fix it (for example if it is possible for you, to somehow peak into the hands of the other players.
